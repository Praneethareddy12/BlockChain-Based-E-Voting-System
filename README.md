# 🗳️ Blockchain-Based E-Voting System

A secure and transparent electronic voting system built using Blockchain technology. This project enables users to cast votes digitally while ensuring vote integrity, transparency, and protection against tampering through blockchain principles.

---

## 🚀 Features

- 🔐 Secure Voting Mechanism
- ⛓️ Blockchain-Based Vote Storage
- 🗳️ Digital Election Management
- 📊 Real-Time Vote Counting
- 👥 User-Friendly Interface
- 🔒 Tamper-Proof Voting Records
- 🌐 Decentralized Voting Concept
- 📈 Transparent Election Results

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Praneethareddy12/BlockChain-Based-E-Voting-System.git
cd BlockChain-Based-E-Voting-System
```

---

## 🔧 Backend Setup

Navigate to backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Start backend server:

```bash
node server.js
```

Backend will run on:

```text
http://localhost:5000
```

---

## 💻 Frontend Setup

Open a new terminal:

```bash
cd blockchain-voting
```

Install dependencies:

```bash
npm install
```

Start frontend application:

```bash
npm run dev
```

Frontend will run on:

```text
http://localhost:5173
```

---

## ▶️ Running the Application

### Start Backend

```bash
cd backend
node server.js
```

### Start Frontend

```bash
cd blockchain-voting
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:5173
```
